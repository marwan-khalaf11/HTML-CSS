# HTML-CSS

## Features Implemented
* **Sticky/Fixed Menu Bar Architecture:** Built with fixed layout positions, custom background filters, contrast adjustments, and responsive interactive transitions on menu element states.
* **Media Overlays:** Integrated custom background auto-streaming looping video structures carrying text headers alongside independent static and fading graphic container spaces.
* **Dynamic Graphics Processing:** Configured explicit graphic component filters (`hue-rotate`, `saturate`, `contrast`), along with circular image clipping configurations.
* **Sized Two-Column Grid Units:** Implemented structurally isolated elements using native padding constraints and explicit dimension properties.
* **Custom Formatted Data Matrix (Table):** Built clean spreadsheet structures containing cell boundary parameters, solid separation guidelines, and distinct target row highlighting.

## Bootstrap Sub-Project Component
In addition to the primary portfolio framework, this repository references our parallel **Bootstrap Sub-Project Framework**. 

### Core Bootstrap Explorations:
1.  **Grid Systems:** Utilizing predefined classes (`.container`, `.row`, `.col-*`) to handle layouts dynamically across different display resolutions.
2.  **Pre-styled UI Kits:** Applying standard Bootstrap components (including alerts, buttons, badges, and responsive cards) to build components quickly.
3.  **Utility Enhancements:** Using responsive utility functions to manage layout metrics without writing bespoke media queries.
